Descripción:

Buscador de datos XML,
Formateado de XML,
Exportador un dato a CSV,
Eliminador de datos en el xml,
Seleccionar días de mora al azar,
Exportar N cantidad de archivos CSV del directorio especificado,
Exportar un solo archivo CSV con N cantidad datos de archivos XML del directorio especificado,
Comparar dos lista de datos CSV y export datos duplicados a archivo CSV,
Unir N cantidad de archivos CSV del directorio especificado,

Search data in xml.
position of arguments:
 main.py file-CSV.csv file-XML.xml XML-file-name.xml

XML format.
position of arguments:
 main.py -f file-XML.xml

Exporting a csv file.
position of arguments:
 main.py -e file-XML.xml file-CSV.csv

Remove values in xml.
position of arguments:
 main.py -r file-CSV.csv file-XML.xml

Select days at random.
position of arguments:
 main.py -rav file-CSV.csv int->number_day

Export N number of CSV files from the specified directory.
position of arguments:
 main.py -export-csv PATH-DIRECTORY

Export a single CSV file with N amount of XML file data from the specified directory.
position of arguments:
 main.py -export-all PATH-DIRECTORY CSV-file-name.csv

Compare two CSV data lists and export duplicate data to CSV file.
position of arguments:
 main.py -compare First-CSV-file-name.csv Second-CSV-file-name.csv

Join N number of CSV files from the specified directory
position of arguments:
 main.py -merge-csv-files PATH-DIRECTORY CSV-file-name.csv

opcional arguments:
 -h, --help  display this help message
 -f, format xml files
 -e, export account statement numbers to csv
 -r, delete statements of account
 -rav, select days at random
 -export-csv, export multiple csv files
 -export-all, export multiple data to csv
 -compare, compares two data lists
 -merge-csv-files, exporting merged data to csv
